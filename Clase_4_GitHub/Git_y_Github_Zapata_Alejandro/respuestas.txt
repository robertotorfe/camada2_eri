¿Quién inventó el sistema de control de versión Git y por qué?
R/ El sistema de control de version Git lo inventó Linus Torvalds para reemplazar a un software similar llamado BitKeeper desarrollado por Larry McVoy. Se hizo principalmente porque este era un software propietario, y porque Linus Torvalds tuvo unas diferencias con Larry McVoy. Git permite no solo mantener un control de versiones, si no que tambien nos permite distribuir y archivar ficheros.


¿A quién pertenece actualmente Github y por qué?
R/ Actualmente Github pertenece a Microsoft. La compra fue realizada para darle entidad al foco en software abierto de Microsoft, y darle llegada a sus herramientas de desarrollo y servicios a nuevas audiencias.
